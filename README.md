# Projeto-faculdade-Cinema
Projeto Final do 1° semestre em C (2019)

O objetivo do projeto é controlar a venda de ingressos, de diversos filmes e diferentes salas, em um cinema. Levando em consideração o benefício de meia entrada para a venda
de ingressos. O projeto apresenta um mapa da sala com cada poltrona marcada corretamente como disponível, reservada ou confirmada (D, R, C).
