# Projeto-faculdade-Cinema
Projeto Final do 1° semestre em C (2019)

Objetivo do projeto: Criar um programa que controle a venda de ingresso, de diversos filmes em diferentes salas. Levando em consideração o benefício de meia entrada para a venda de ingressos. O projeto apresenta um mapa das poltronas da sala, com cada poltrona marcada como disponível, confirmada ou reservada (D, C, R).
